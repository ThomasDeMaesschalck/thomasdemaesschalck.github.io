About me

After being self-employed for the bulk of the past sixteen years, I'm now making the transition being a software developer. Over the years, I've mainly done web publishing and online marketing work, plus I've also dabbled into writing in-depth freelance articles for a financial website.

My journey to become a software developer started three years ago, when I enrolled in evening school to get an associate degree (HBO5) in computer programming.

After learning the basics of object oriented programming, I made several fun projects including an Arduino-based line follower, a desktop Java application that simulates a bar, a Java EE + JSF-based crowdfunding website, a retro-style Connect Four game that uses Spring Boot microservices + Angular, and a Spring Boot microservices and Angular-based final project.

I'm looking forward to start the Switchfully track in September 2021 to hone my skills as a full stack developer.
